Google Translate Chat



A chat application where every message is translated to a chosen language and then translated back to the original. The users will be able to see how close/divergent the translated messages compare to the original and can be a fun exercise to see which language combinations yield the most accurate translations.


    The team is currently looking for individuals who would like to work on one of the following categories:
  1. Technical (Front-end development)
  2. Technical (Back-end development)
  3. Technical (Mobile app development)

The Multilingual Newspaper



This project aims to be a platform where intermediate learners of a language are able to practice while being immersed in a cultural context. News articles from the target country will be presented in a list view from which users are able to select one to read. While reading the article, if the user stumbles upon a word that they do not understand, they are able to click on it to be presented with the translation.


    The team is currently looking for individuals who would like to work on one of the following categories:
  1. Technical (Front-end development)
  2. Technical (Back-end development)

Automated Conversational Partner Matching



This project aims to be a platform where natives can be paired with learners depending on their current proficiency level. In learning a language, the most important part is consistent practice while immersed in the language. There is no better way to achieve this than to speak with natives.
