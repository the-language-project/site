Web Applications

The Multilingual Newspaper



This project aims to be a platform where intermediate learners of a language are able to practice while being immersed in a cultural context. News articles from the target country will be presented in a list view from which users are able to select one to read. While reading the article, if the user stumbles upon a word that they do not understand, they are able to click on it to be presented with the translation.

Luri Language Archive


Tags: Luri

The Luri language is a dying language in need of public awareness. Thus, the Luri Language Archive will serve as a central place for Luri texts/resources and a way to promote the language in a multitude of ways. Details have yet to be decided upon, but the team will be working to improve this existing website: https://lurishacademy.org/"
